學習心得 | 台大計資中心 Python程式設計入門

  2021-08-31   學習心得   python 0

本身寫程式經驗

  1. 統計分析修課&工作使用SAS、R，擁有SAS base programming認證
  2. 兩學期程式設計(java)
  3. 一學期網頁設計美學(HTML、CSS、javascript)

前言

想學Python很久了，今年暑假決定從台大計資中心紀俊男老師的Python程式設計入門入手。網路上相關課程詳細心得一直很少，決定上來分享上課心得，第一次使用HackMD寫網誌，再請大家多多指教:-)

課程內容

  • 上課網址：https://webpageprodvm.ntu.edu.tw/e-learning/News_Content.aspx?n=72682&s=78789
  • Lecture
  • Requirement
    • 作業：70%
    • 簽到：30%

來說說這堂課的優缺點

  • 優點
    • 上課教得很仔細、程式運行方式和原理會用簡單的方式解釋得很清楚
    • 作業簡單易寫，老師提示完作業就差不多了
  • 缺點
    • 對於已有寫程式經驗的人來說，許多概念如條件式、迴圈邏輯已經有基本知識，這門課老師會從頭開始解釋，比較適合完全沒有寫程式經驗的新手 (所以我都在直播時簽到完，等課程影片上傳直接1.5倍速上課)
    • 無專案實作，少了累積作品集的機會

後記

課程結束一個半月後通過名單公布在官網，有成績才能申請證書，證書印製好後不會另行通知，兩週工作天後需行到計中一樓諮詢櫃檯領取
